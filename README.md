# ToDo-приложение

## Описание
Чистый ToDo-лист с правильной архитектурой. Демонстрация разделения логики и UI.

## Технологии
- HTML
- CSS
- Vanilla JS

## Инструкция по запуску
Открыть index.html в браузере.

## Архитектура
Приложение организовано по принципу разделения ответственностей:

- **UI (User Interface)**: Отвечает за отображение элементов на странице. Работает с DOM напрямую.
- **Controller**: Обрабатывает пользовательские события (клик по кнопке, ввод текста). Связывает UI и Core.
- **Core (Бизнес-логика)**: Основная логика приложения - добавление, удаление, изменение статуса задач.
- **Storage**: Отвечает за хранение данных (в данном случае - в localStorage браузера).
